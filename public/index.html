Merceria L'Intimo di Paola API

    Routes:

  • /prodotti/{genere}(donna|uomo|kids)/{categoria}(maglie|intimo|canottiere|vestaglie|pigiami|calze)

    GET Request = Get all products by gender and type

    POST Request = Post new product with following data in the body of the request: pic_src, name, category, gender, brand, material, color, size, price, description.

  • /prodotti/{genere}(donna|uomo|kids)/{categoria}(maglie|intimo|canottiere|vestaglie|pigiami|calze)/{id}

    GET Request = Get one product by gender,type and id

    DELETE Request = Delete one product of a category, gender by its id
