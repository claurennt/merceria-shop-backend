Merceria L'Intimo di Paola API

    Routes:

  • /prodotti/{genere}(donna|uomo|kids)/{categoria}(maglie|intimo|canottiere|vestaglie|pigiami|calze)

    GET Request = Get all products by gender and type

    DELETE Request = Delete multiple products: if no codition is passed, all products from categoria and genere passed in the url will be deleted; if there is a key:value pair condition, only products matching categoria genere and key:value will be deleted

  • /prodotti/{categoria}(maglie|intimo|canottiere|vestaglie|pigiami|calze)/{id}

    GET Request = Get one product by category and id

    DELETE Request = Delete one product of a category, gender by its id

    PUT Request = Update certain fields, or all through req.body

  • /prodotti/aggiungi

    GET Request = Displays form to upload new product to database

    POST Request = Posts new product via form

    • POST Request = Post new product with following data in the body of the request: pic_src, name, category, gender, brand, material, color, size, price, description.
